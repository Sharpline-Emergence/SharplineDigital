# Sharpline Digital — Website

Business website for Sharpline Digital, a web design & development company serving small businesses in British Columbia's Fraser Valley.

## Project Structure

```
sharplinedigital/
├── index.html          # Homepage
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── main.js         # Mobile nav, scroll behaviour, animations
├── images/             # Add client photos and assets here
└── README.md
```

## Colours

| Name  | Hex       | Usage                        |
|-------|-----------|------------------------------|
| Teal  | `#2ec4a9` | Primary accent, CTAs         |
| Navy  | `#0a1628` | Backgrounds, headings        |
| White | `#ffffff` | Surfaces, body text bg       |

## Fonts

- **Syne** (800) — headings and logo
- **DM Sans** (300, 400, 500) — body copy

Both loaded via Google Fonts.
